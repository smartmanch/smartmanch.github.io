# smartmanch.github.io
स्मार्ट मंच राजस्थान शिक्षा विभाग के सरकारी स्कूलों के विद्यार्थियों के लिए तैयार किया गया एक अभिनव ई-लर्निंग पोर्टल है। इसका मुख्य उद्देश्य शिक्षा को डिजिटल माध्यम से रोचक, इंटरैक्टिव और सुलभ बनाना है।इस पोर्टल की परिकल्पना और निर्माण श्री रोहिताश कुमार भड़िया (प्रधानाचार्य) द्वारा किया गया है।
